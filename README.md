# IPL-Bowler-Analysis
An end-to-end Python &amp; SQL pipeline using Pandas to clean multi-GB ball-by-ball IPL data. Implemented an in-memory SQLite layer (sqlite3) to crunch death-overs metrics (Overs 16-20) with a consistency filter (\ge 120 balls). Synthesized Matplotlib charts to rank top 10 bowlers based on lowest economy rates and highest dot-ball %.
